Setup phase 1

Client Realm server
Introduction
PCKT_R_WELCOME

Update the client
(can be skipped although it's not recommended)
PCKT_C_VERSION
PCKT_R_DOWNLOAD
as many times as needed, optional
PCKT_R_DOWNLOAD_EOF

Pick up a server
PCKT_C_GETWORLDLIST
PCKT_R_WORLD
as many times as needed, optional
PCKT_R_WORLD_EOF
* Close socket *

Setup phase 2

Client World server
Introduction
PCKT_W_WELCOME

Authentication
PCKT_C_AUTH
PCKT_W_AUTH_ACK

Character list
PCKT_C_GETCHARLIST
PCKT_W_CHARLIST_ADD
as many times as needed, optional
PCKT_W_CHARLIST_EOF

Character management
(this step is optional, there's two choices possible)
(1) PCKT_C_CREATE_CHAR
PCKT_W_CREATE_CHAR_ACK
(2) PCKT_C_DELETECHAR
(returns nothing)

Prepare to enter the world
(this packet should be sent only if required)
PCKT_C_KICK_GHOST

Then enter the world
PCKT_C_ENTER_WORLD
PCKT_W_ENTER_WORLD_ACK
PCKT_W_PLAYER_DATA
